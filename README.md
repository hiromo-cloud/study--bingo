# ✨ STUDY BINGO

お子様の学習習慣を楽しくサポートする、ビンゴ型モーニングルーティン・学習管理アプリです。

## 🚀 特徴
- **カスタマイズ可能な項目**: お子様の学習内容に合わせてビンゴの項目を自由に変更可能。
- **カラーテーマ機能**: 「さくら」「そら」「わかば」など、気分に合わせて色を選べます。
- **タイマー機能**: 制限時間内に終わらせる達成感を演出。
- **ビンゴ演出**: 揃った時の華やかなエフェクトでモチベーションをアップ！

## 🛠 開発・実行方法
1. `npm install` で依存関係をインストール
2. `npm run dev` でローカルサーバーを起動
3. `npm run build` で本番用ファイルを生成
